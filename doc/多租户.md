# 系统端

# 管理端

创建并管理`bucket`。

通过邀请方式允许其他用户访问`bucket`。

# 用户端

选择进入有访问权限的`bucket`。

如果用户有唯一的`bucket`，自动选择这个`bucket`进入，否则要求用户从可用列表中选择一个`bucket`。

在编译阶段，过将环境变量`VUE_APP_TMW_REQUIRE_BUCKET`设置为`yes`或`true`打开多租户模式。

# 关闭多租户模式

通过环境变量，关闭多租户模式。

# 单用户模式和多用户模式

单用户模式：存储空间独享。

多用户模式：存储空间共享。
