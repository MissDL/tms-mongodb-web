# 空间

## 邀请用户访问空间

POST /bucket/coworker/invite

查询参数

| 参数   | 说明             |
| ------ | ---------------- |
| bucket | 空间名称（name） |

POST 参数

| 参数     | 说明 |
| -------- | ---- |
| nickname |      |

返回结果

```
{"msg":"正常","code":0,"result":"ok"}%
```

## 删除授权访问用户

GET /bucket/coworker/remove

查询参数

| 参数     | 说明             |
| -------- | ---------------- |
| bucket   | 空间名称（name） |
| coworker | 授权用户的 id    |
